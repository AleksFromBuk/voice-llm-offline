

## Каталог `models/`

Здесь хранятся **локальные модели**, которые не входят в репозиторий
(слишком большие по размеру).

Сейчас приложению нужна **одна Vosk-модель для русского языка**:

## 1. Установка Vosk-модели

1. Скачайте модель Vosk для русского языка:
- `vosk-model-small-ru-0.22` по ссылке https://alphacephei.com/vosk/models/vosk-model-ru-0.22.zip

2. Скачайте русскую модель, например:

   - `vosk-model-small-ru-0.22` (рекомендуется — 'компактная и быстрая':) )

3. Распакуйте архив в папку `models/` вашего проекта:

```text
voice-llm-offline/
  models/
    vosk-model-small-ru-0.22/
      am/
      conf/
      graph/
      ivector/
      ...
````

В результате путь к модели должен быть:

```text
models/vosk-model-small-ru-0.22
```

Именно это имя ожидает код в `app.py`
(метод `_load_models()`).

Если вы используете другую русскую модель Vosk,
можно либо:

* переименовать директорию под `vosk-model-small-ru-0.22`,
  либо
* изменить путь в `app.py`:

```python
model_path = os.path.join("models", "vosk-model-small-ru-0.22")
```

заменив `vosk-model-small-ru-0.22` на имя вашей папки.

## 2. Почему модели не в GitHub

Vosk-модели весят от сотен мегабайт до единиц гигабайт, поэтому:

* они **не коммитятся** в репозиторий;
* по умолчанию каталог `models/` добавлен в `.gitignore`.

Каждый разработчик/пользователь скачивает и распаковывает их локально.
